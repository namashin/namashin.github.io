2021.4 - 2023.6

ソフトウェアエンジニア - Pixel Software 株式会社

工業用リアルタイムプロセス情報管理システム（PLANETMEISTER）およびそのゲートウェイソフトウェア開発、バグ修正を担当しました.

このプロジェクトでは、TCP/IPプロトコルを使用してセンサーからデータを受信し、効率的かつ信頼性の高い処理を実現しました.

なぜTCP/IPを選んだかというと、センサーから送信されるデータは、欠落や順序の乱れが許されない重要な情報でした.

そのため、データ通信の信頼性を重視し、データの確実な送信を保証するTCPプロトコルを採用しました.

TCPは、通信中にデータの損失が発生した場合の再送機能や、送信順序の保持が可能であり、リアルタイム性と安定性の両立を実現するための最適な選択でした.

具体的な実装としては、センサーはTCPを介してデータを送信し、サーバーサイドでTCPソケットを使用してリアルタイムでデータを受信しました.

受信したデータパケットを解析し、センサーIDや測定値など必要な情報を抽出しました.

また、Golangの並行処理機能（goroutinesとchannels）を活用して大量の接続を効率的に処理し、スケーラビリティを確保しました.

主な成果としては、データスループットを40%向上, 処理時間を大幅に短縮し、リアルタイム性を実現, 信頼性の高いデータ通信により、旧システム（VBやC言語で構築）からのスムーズな移行を達成.

このプロジェクトを通じて、TCP/IPプロトコルを活用した信頼性の高い通信と、Golangによる高効率な並行処理を組み合わせ、システム全体のパフォーマンスと安定性を大幅に向上させることができました.

また、C#による制御ソフトウェアの開発を通じて、操作員がリアルタイムで産業データを監視できる仕組みを提供しました.

7人チームの一員として、設計からテストまで一連の開発工程に携わり、高品質なソフトウェアの納品に貢献しました .

东方航空 派工最適化プロジェクト

Overview

东方航空のタスク割り当てとスケジューリング効率を向上させるための派工最適化プロジェクトに参画。深層強化学習（CVNet）と群知能アルゴリズム（ABC、FA）を活用し、リアルタイムで動的な環境に適応可能な派工モデルを開発

Roles

【モデル設計】 ・タスク割り当てを半マルコフ決定プロセス（SMDP）としてモデリングし、タスクの優先度、移動コスト、スキル適合性を考慮 ・従業員のスキル（A、B、C）に基づく組み合わせ制約を解決 【アルゴリズム実装】 ・CVNetを用いてタスクの価値関数を学習し、動的なタスク割り当てを最適化 ・群知能アルゴリズム（ABC、FA）を活用して移動コストを削減しつつスケジュール効率を向上 【成果提案】 ・プロジェクトを通じて得られた研究成果を东方航空に提案。アルゴリズムの実装方法、パフォーマンス評価、および実運用での適用可能性を含む

Results

【成果】 ・东方航空の運用効率向上に寄与する新しい派工モデルを提案 ・リアルタイムのタスク動的調整が可能なアルゴリズムを開発 ・提案内容は航空運用現場での実用化を前提に設計
