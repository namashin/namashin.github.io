2024.9 - ปัจจุบัน

โครงการปรับปรุงการจัดสรรงาน (Dispatch Optimization) ของ China Eastern Airlines

Overview

เข้าร่วมโครงการปรับปรุงกระบวนการจัดสรรงานของสายการบิน China Eastern Airlines เพื่อเพิ่มประสิทธิภาพการมอบหมายงานและการจัดตารางเวลา โดยใช้เทคนิค Deep Reinforcement Learning (CVNet) และอัลกอริทึม Swarm Intelligence (ABC, FA) เพื่อพัฒนารูปแบบการจัดสรรงาน (dispatch model) ที่สามารถปรับเปลี่ยนตามสภาพแวดล้อมแบบเรียลไทม์ได้

Roles

[การออกแบบโมเดล] - สร้างโมเดลการจัดสรรงานในรูปแบบ Semi-Markov Decision Process (SMDP) โดยคำนึงถึงลำดับความสำคัญของงาน (task priority), ค่าใช้จ่ายในการเดินทาง (travel cost) และความสอดคล้องของทักษะ (skill compatibility) - จัดการข้อจำกัดด้านการผสมผสานตามระดับทักษะ (A, B, C) ของพนักงาน [การพัฒนาอัลกอริทึม] - ใช้ CVNet ในการเรียนรู้ value function ของงาน เพื่อให้การจัดสรรงานแบบไดนามิกมีประสิทธิภาพสูงสุด - ประยุกต์ใช้อัลกอริทึม Swarm Intelligence (ABC, FA) ในการลดต้นทุนด้านการเดินทาง และเพิ่มประสิทธิภาพการจัดตารางเวลา [การนำเสนอผลลัพธ์] - นำเสนอผลงานวิจัยให้กับ China Eastern Airlines ซึ่งรวมถึงรายละเอียดการพัฒนาอัลกอริทึม การประเมินประสิทธิภาพ และความเป็นไปได้ในการใช้งานจริง

Results

[ผลลัพธ์] - เสนอโมเดลการจัดสรรงานรูปแบบใหม่ ที่ช่วยเพิ่มประสิทธิภาพในการดำเนินงานของ China Eastern Airlines - พัฒนาอัลกอริทึมที่สามารถปรับตัวต่อการเปลี่ยนแปลงแบบเรียลไทม์ได้ - ออกแบบโซลูชันให้เหมาะสมกับการใช้งานในสถานการณ์จริงภาคสนาม
